# Kanban Tools

* Tools that help you discover more about your Kanban system.
* Currently very Jira-specific but more to follow

## Jira

### Pre-requisites

* ruby 2.1.x (might work with 1.9.x)
* [jira-ruby 0.1.13](https://github.com/sumoheavy/jira-ruby) (or greater)
* [table_print 1.5.3](http://tableprintgem.com) (or greater)

### Tools

#### History

Usage:

    ./transition_history.rb -v <query>
